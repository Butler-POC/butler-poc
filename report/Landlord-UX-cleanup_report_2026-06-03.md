# 임대인 앱 화면 UX 정리 — 작업 보고서

- 작성일: 2026-06-03
- 작업자: StabAn-NRH
- 범위: Butler Landlord 앱 전반의 군더더기 정리(기능코드·메뉴·로그아웃·버튼 형태)

---

## 1. 배경

메인화면·드로어 도입 후, 개발용 잔재(기능코드 노출)와 중복 진입점(상단 로그아웃, 공통기능 메뉴)을 사용자 관점으로 정리했다.

## 2. 변경 내용

### 2-1. 기능코드(L-01 등) 전면 비노출
- 사용자에게 보일 필요 없는 개발용 식별자다. 임대인 화면 5곳(`내 건물`, `새 건물 등록`, `건축물대장`, `임차인 관리`, `공실 등록`)의 eyebrow에서 `· L-0X` 제거 → `임대인`만 표시.
- 드로어(`NavDrawer`)의 기능 목록에서도 코드 표기 제거(기능명만).

### 2-2. 드로어/로그아웃 일원화
- 상단바(`TopBar`) 우측의 **로그아웃 버튼 제거** → 로그아웃은 드로어 하단 한 곳으로 통일.
- 드로어 기능 리스트 상단의 "임대인 기능" 그룹 타이틀 제거(간결화).

### 2-3. 공통 기능(C-01 문서추출 · C-02 챗봇) 메뉴 숨김
- 공통기능은 다른 기능(L-06 법률·T-02 하자)을 위한 **엔진/인프라**여서, 사용자에게 별도 진입 메뉴로 노출하지 않기로 함.
- **네비게이션 항목만** 제거(드로어 공통 그룹 + 임차인·중개업자 홈의 공통 섹션). 화면·라우트·API·서비스 등 **구현 코드는 모두 보존** → L-06/T-02 등은 챗봇 엔진을 그대로 사용.

### 2-4. 건물 카드 내 임차인 항목 형태
- "임차인 관리"를 박스 버튼 → 근저당권 구분선과 동일한 **상단 구분선 텍스트 행**으로 변경(시각적 군더더기 축소).

## 3. 주요 파일

- `components/layout/TopBar.vue` — 로그아웃 제거
- `components/layout/NavDrawer.vue` — 코드/공통그룹/타이틀 제거
- `components/domain/RoleHome.vue` — 공통 섹션 제거(임차인·중개업자 홈 공용)
- `lib/roleFeatures.ts` — 기능 라벨/링크 관리(코드는 데이터로만 보존)
- `views/landlord/*` — eyebrow 코드 제거
- `components/domain/BuildingCard.vue` — 임차인 항목 형태

## 4. 비고

- 공통기능 화면/라우트는 살아 있으므로, 추후 필요 시 메뉴를 다시 노출하면 즉시 복구된다(코드 변경 없음).
